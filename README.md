# Golf Shot Classification Using Deep Learning

This project builds a neural network model to classify golf shots as good or poor based on launch monitor style data.

The dataset was synthetically generated to simulate realistic golf performance variables such as club speed, launch angle, spin rate, and carry distance.

The model used is a feedforward neural network (MLP) trained using TensorFlow/Keras.

To run:
1. Open the notebook file.
2. Run cells in order.
